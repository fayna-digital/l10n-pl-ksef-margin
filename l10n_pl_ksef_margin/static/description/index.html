Odoo 17 Module

Fayna KSeF

Integracja z KSeF 2.0 (FA 3) dla Odoo 17. Wysyłka faktur, monitorowanie statusów i pobieranie UPO — z pełną obsługą VAT Marża dla biur turystycznych.

Rozwiązanie: Fayna Digital dla CampScout Autor: Volodymyr Shevchenko Schemat: FA(3) v1-0E Wersja: 17.0.2.2.6
Możliwości

Co umie moduł

📤

Wysyłka do KSeF

Generowanie XML FA(3) i wysyłka przez API KSeF 2.0 jednym kliknięciem bezpośrednio z faktury.

🏷️

VAT Marża turystyczna

Pełna obsługa faktur marżowych: P_13_11, PMarzy, mieszane faktury marża + regularny VAT.

↩️

Faktury korygujące (KOR)

Automatyczne budowanie sekcji DaneFaKorygowanej z NrKSeFFaKorygowanej dla not kredytowych.

🔄

Monitoring statusów

Cron co 15 minut sprawdza statusy sesji KSeF i automatycznie aktualizuje Nr KSeF po akceptacji.

📄

UPO jako załącznik

Urzędowe Potwierdzenie Odbioru pobierane z KSeF i zapisywane bezpośrednio do faktury w Odoo.

📊

Dashboard KSeF

Przegląd faktur: zaakceptowane, oczekujące, odrzucone — z wykresem aktywności i filtrowaniem.

📥

Bufor faktur przychodzących

Codzienna synchronizacja faktur dostawców z KSeF do bufora w Odoo z automatycznym cron.

⚡

Masowa wysyłka

Wysyłka wielu faktur naraz przez wizard — z podglądem wyników i obsługą błędów per faktura.

Przepływ

Jak działa wysyłka

1
Potwierdź fakturę w Odoo

Zwykła faktura klienta (out_invoice) lub nota kredytowa (out_refund). Status: Opublikowana.

2
Kliknij „Wyślij do KSeF"

Moduł buduje XML FA(3), otwiera sesję interaktywną i wysyła zaszyfrowany dokument przez API KSeF.

3
Status: Waiting for Status

KSeF przetwarza dokument. Nr referencyjny sesji (SO-...) zostaje zapisany na fakturze.

4
Status: Accepted — Nr KSeF nadany

Cron co 15 minut odpytuje KSeF. Po akceptacji Nr KSeF zostaje wpisany do faktury, UPO dostępne do pobrania.

Wymagania

Instalacja i konfiguracja

1
Zależności Odoo

Moduły account i l10n_pl muszą być zainstalowane.

2
Biblioteka ksef-client

pip install ksef-client>=0.8 — obsługa KSeF 2.0 API z szyfrowaniem RSA-OAEP + AES-256-CBC.

3
Konfiguracja firmy

W Ustawienia → Firma → zakładka KSeF: NIP, token autoryzacyjny KSeF oraz certyfikat szyfrowania.
